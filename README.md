## 基于 ESP8266、Python 和 Flask 实现的智能车库管理系统 

> 为什么是智能，哪里智能了？ 
> 我也不知道，谁让我的毕业设计选题里有个智能！  

### 环境   

- Arduino: ESP8266 代码编辑、编译和烧录  

- Python: 服务端应用实现 

   - Flask 实现用户端 Web 访问  

   - Socket 与 ESP8266 进行通信  

### 运行  

1. 画 PCB 焊接硬件电路  

2. PC 或其他任何设备上运行服务端  

3. ESP8266 烧录，插电，运行  

4. 浏览器访问服务端
